## King's Raid Index

This is a Site consolidating information on the Game 'King's Raid'. It is still very much wip.

## Docker Setup

`docker run --name YOURMONGODBNAME --restart=always -d -p 27017:27017 mongo`

# How to start

## Frontend

1. `cd Client | yarn`
2. `yarn start`

## Backend

1. `cd Server | yarn`
2. `yarn start`
